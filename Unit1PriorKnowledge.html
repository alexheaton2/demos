The Unit Circle: Your Trigonometric Compass

The Key Idea

The unit circle is your best friend for quickly evaluating trigonometric functions. It's a circle with a radius of 1 centered at the origin. For any angle $\theta$, the point $(x, y)$ where the angle's terminal side intersects the circle gives you the cosine and sine values:

  • $\cos(\theta) = x$-coordinate
  • $\sin(\theta) = y$-coordinate

Knowing the coordinates for key angles (like $\pi/6, \pi/4, \pi/3$ and their multiples) is essential for evaluating limits and understanding trig function behavior without a calculator.

Function Domains & Continuity

The Key Idea

A function is continuous on its domain. If a question asks where a function is continuous, it's often a disguised question about its domain. Remember the domain restrictions for common functions:

  • Logarithms: $log(x)$ is only defined for $x > 0$.
  • Trigonometry: Functions like $tan(x)$ and $cot(x)$ have vertical asymptotes where their denominators are zero. For $cot(x) = \frac{cos(x)}{sin(x)}$, the domain excludes values where $sin(x) = 0$ (i.e., multiples of $\pi$).
  • Square Roots: $\sqrt{x}$ is only defined for $x \ge 0$.

Sample Problem

Which of the following functions are continuous for all real numbers?
I. $h(x) = log(x)$
II. $g(x) = cot(x)$

Solution Walkthrough:

1. Analyze $h(x) = log(x)$. The domain of the natural logarithm is $(0, \infty)$. Since this is not all real numbers, $h(x)$ is not continuous for all real numbers.

2. Analyze $g(x) = cot(x)$. The function $cot(x)$ has vertical asymptotes at $x = k\pi$ for any integer $k$ (e.g., at $x=0, \pi, 2\pi, ...$). Since it has discontinuities, it is not continuous for all real numbers.

Answer: Neither h nor g.

Factoring & Cancelling

The Key Idea

Direct substitution into a limit often yields the indeterminate form $\frac{0}{0}$. This is a huge clue that you should try to factor the numerator and denominator to find a common factor that can be cancelled. A classic pattern to watch for is the **Difference of Squares**: $a^2 - b^2 = (a-b)(a+b)$.

Sample Problem

Find $\lim_{x\rightarrow -4}\frac{x^{2}-16}{x+4}$.

Solution Walkthrough:

1. Try direct substitution: $\frac{(-4)^2 - 16}{-4+4} = \frac{16-16}{0} = \frac{0}{0}$. Indeterminate form! Time to factor.

2. Factor the numerator using difference of squares: $x^2 - 16 = (x-4)(x+4)$.

3. Rewrite the limit and cancel: $\lim_{x\rightarrow -4}\frac{(x-4)(x+4)}{x+4} = \lim_{x\rightarrow -4}(x-4)$.

4. Now, use direct substitution on the simplified expression: $-4 - 4 = -8$.

Answer: -8.

Limits & Infinity

The Key Idea

Understanding how fractions behave with very large or very small numbers is crucial. Think about these concepts:

  • $\frac{\text{Number}}{\text{Huge Number}} \rightarrow 0$ (e.g., $\lim_{x\rightarrow\infty} \frac{1}{x} = 0$)
  • $\frac{\text{Number}}{\text{Tiny Positive Number}} \rightarrow +\infty$ (e.g., $\lim_{x\rightarrow 1^+} \frac{1}{x-1}$)
  • $\frac{\text{Number}}{\text{Tiny Negative Number}} \rightarrow -\infty$ (e.g., $\lim_{x\rightarrow 1^-} \frac{1}{x-1}$)

For limits at infinity of rational functions, compare the degrees of the numerator and denominator. If the degree of the numerator is higher, the limit is unbounded ($\pm\infty$).

Sample Problem

Find $\lim_{x\rightarrow\infty}\frac{x^{2}-4}{x+4}$.

Solution Walkthrough:

1. Identify the highest power of $x$ in the numerator (2) and the denominator (1).

2. Since the degree of the numerator is greater than the degree of the denominator, the function grows without bound as $x$ approaches infinity.

3. To be more formal, divide every term by the highest power in the denominator ($x$): $\lim_{x\rightarrow\infty}\frac{x - \frac{4}{x}}{1 + \frac{4}{x}}$. As $x \rightarrow \infty$, the terms $\frac{4}{x}$ go to 0, leaving $\frac{x}{1}$, which goes to $\infty$.

Answer: The limit is unbounded.

One-Sided Limits

The Key Idea

A one-sided limit asks what value a function is approaching as $x$ gets closer to a number from a specific direction.

  • From the right ($x \rightarrow c^+$): Use values slightly greater than $c$. For $x \rightarrow 1^+$, think $x=1.1$ or $x=1.001$.
  • From the left ($x \rightarrow c^-$): Use values slightly less than $c$. For $x \rightarrow 1^-$, think $x=0.9$ or $x=0.999$.

This is especially important for vertical asymptotes and for understanding tables of values.

Sample Problem 1: From a Formula

Let $f(x) = -\frac{4}{x-1}$. Describe the one-sided limits at $x=1$.

Solution Walkthrough:

1. **Limit from the right ($x \rightarrow 1^+$):** Let's plug in a number just bigger than 1, like $x=1.001$. The denominator is $1.001 - 1 = 0.001$, which is a tiny positive number. So we have $-\frac{4}{\text{tiny positive}} \rightarrow -\infty$.

2. **Limit from the left ($x \rightarrow 1^-$):** Let's plug in a number just smaller than 1, like $x=0.999$. The denominator is $0.999 - 1 = -0.001$, which is a tiny negative number. So we have $-\frac{4}{\text{tiny negative}} \rightarrow +\infty$.

Answer: $\lim_{x\rightarrow 1^+}f(x) = -\infty$ and $\lim_{x\rightarrow 1^-}f(x) = +\infty$.

Sample Problem 2: From a Table

The table gives values of $g(x)$. What is a reasonable estimate for $\lim_{x\rightarrow -8^+} g(x)$?

x g(x)
-8.025-5.2
-8.005-5.04
-8.001-5.008
-8?
-7.9990.008
-7.9950.04
-7.9750.2

Solution Walkthrough:

1. "From the right" ($x \rightarrow -8^+$) means we look at x-values that are slightly *greater* than -8. On a number line, these are numbers like -7.999, -7.995, etc.

2. Look at the corresponding $g(x)$ values for $x > -8$: As $x$ gets closer to -8 (from -7.975 to -7.999), $g(x)$ gets closer to 0 (from 0.2 to 0.008).

Answer: 0.

The Conjugate Method

The Key Idea

When you see a limit with a square root that gives $\frac{0}{0}$, think "conjugate." This is just a fancy application of the difference of squares formula. The conjugate of $\sqrt{a} - b$ is $\sqrt{a} + b$. Multiplying them together gets rid of the square root: $(\sqrt{a}-b)(\sqrt{a}+b) = (\sqrt{a})^2 - b^2 = a - b^2$.

Remember to multiply both the numerator AND the denominator by the conjugate to keep the expression's value the same.

Sample Problem

Find $\lim_{x\rightarrow 4}\frac{2-\sqrt{4x-12}}{x-4}$.

Solution Walkthrough:

1. Direct substitution gives $\frac{2-\sqrt{16-12}}{4-4} = \frac{2-\sqrt{4}}{0} = \frac{2-2}{0} = \frac{0}{0}$. Time for the conjugate.

2. The conjugate of the numerator is $2+\sqrt{4x-12}$. Multiply the top and bottom by it.

3. $\lim_{x\rightarrow 4}\frac{2-\sqrt{4x-12}}{x-4} \cdot \frac{2+\sqrt{4x-12}}{2+\sqrt{4x-12}} = \lim_{x\rightarrow 4}\frac{4 - (4x-12)}{(x-4)(2+\sqrt{4x-12})}$

4. Simplify the numerator: $4 - 4x + 12 = 16 - 4x = -4(x-4)$.

5. Cancel the $(x-4)$ term: $\lim_{x\rightarrow 4}\frac{-4(x-4)}{(x-4)(2+\sqrt{4x-12})} = \lim_{x\rightarrow 4}\frac{-4}{2+\sqrt{4x-12}}$

6. Now substitute $x=4$: $\frac{-4}{2+\sqrt{16-12}} = \frac{-4}{2+\sqrt{4}} = \frac{-4}{2+2} = \frac{-4}{4} = -1$.

Answer: -1.

Intermediate Value Theorem

The Key Idea

The Intermediate Value Theorem (IVT) states that if a function $g$ is continuous on a closed interval $[a, b]$, then it must take on every y-value between $g(a)$ and $g(b)$.

The key is to distinguish between domain values (the x-values, or inputs) and range values (the y-values, or outputs). The theorem guarantees a y-value exists, and the corresponding x-value (c) is in the domain interval.

Sample Problem

Let $g$ be a continuous function on the closed interval $[-1, 3]$, where $g(-1)=2$ and $g(3)=-5$. Which of the following is guaranteed by the Intermediate Value Theorem?

  • $g(c) = -7$ for at least one c between -1 and 3
  • $g(c) = 0$ for at least one c between -1 and 3
  • $g(c) = -3$ for at least one c between -5 and -2
  • $g(c) = 0$ for at least one c between -5 and 2

Solution Walkthrough:

1. Identify the domain interval (the x-values): $[-1, 3]$.

2. Identify the range of y-values at the endpoints: from $g(-1)=2$ to $g(3)=-5$. This means the function must hit every y-value between -5 and 2.

3. The IVT guarantees that for any y-value $k$ in the interval $[-5, 2]$, there is an x-value $c$ in the interval $[-1, 3]$ such that $g(c)=k$.

4. Let's check the options:

  • Is $g(c)=-7$ guaranteed? No, -7 is not between -5 and 2.
  • Is $g(c)=0$ guaranteed? Yes, 0 is between -5 and 2, and the theorem says the c-value will be between -1 and 3. This fits perfectly.
  • The last two options confuse the domain and range intervals. The c-value must be in the domain interval $[-1, 3]$.

Answer: $g(c)=0$ for at least one c between -1 and 3.

Exponent Rules

The Key Idea

Exponent rules are essential for simplifying expressions, especially with the number $e$. Key rules include:

  • Negative Exponent: $x^{-a} = \frac{1}{x^a}$. For example, $e^{-1} = \frac{1}{e}$.
  • Product Rule: $x^a \cdot x^b = x^{a+b}$.
  • Quotient Rule: $\frac{x^a}{x^b} = x^{a-b}$.

Gut Check!

If you forget a rule, test it with simple numbers. Is $2^5 / 2^3$ equal to $2^{5-3}=2^2=4$ or $2^{5/3}$? The first one is correct ($32/8=4$). This can save you from a mistake!

Sample Problem

Let $f(x)=\frac{x+1}{xe^{x}+e^{x}}$ when $x\ne-1$. If $f$ is continuous, find $f(-1)$.

Solution Walkthrough:

1. For $f$ to be continuous, $f(-1)$ must be equal to $\lim_{x\rightarrow -1} f(x)$. We need to find this limit.

2. Direct substitution gives $\frac{0}{0}$. We need to simplify. Factor the denominator: $xe^x + e^x = e^x(x+1)$.

3. Rewrite and cancel: $\lim_{x\rightarrow -1} \frac{x+1}{e^x(x+1)} = \lim_{x\rightarrow -1} \frac{1}{e^x}$.

4. Now substitute $x=-1$: $\frac{1}{e^{-1}}$.

5. Use the negative exponent rule: $\frac{1}{e^{-1}} = e^1 = e$.

Answer: $e$.

Trig Identities in Limits

The Key Idea

Just like with factoring, if you get $\frac{0}{0}$ in a trig limit, look for an identity to simplify the expression. A common one is the **double angle formula for sine**: $sin(2x) = 2sin(x)cos(x)$.

Sample Problem

Find $\lim_{x\rightarrow \frac{\pi}{2}}\frac{sin(2x)}{cos(x)}$.

Solution Walkthrough:

1. Direct substitution: $\frac{sin(\pi)}{cos(\pi/2)} = \frac{0}{0}$. Time for an identity.

2. Apply the double angle formula to the numerator: $sin(2x) = 2sin(x)cos(x)$.

3. Rewrite the limit and cancel: $\lim_{x\rightarrow \frac{\pi}{2}}\frac{2sin(x)cos(x)}{cos(x)} = \lim_{x\rightarrow \frac{\pi}{2}} 2sin(x)$.

4. Now substitute $x=\frac{\pi}{2}$: $2sin(\frac{\pi}{2}) = 2(1) = 2$.

Answer: 2.

Limits with Absolute Value

The Key Idea

The key to limits involving $\sqrt{x^2}$ is to remember that $\sqrt{x^2} = |x|$. The absolute value function is piecewise:

$|x| = \begin{cases} x & \text{if } x \ge 0 \\ -x & \text{if } x < 0 \end{cases}$

This is CRITICAL for limits to $-\infty$. When $x$ is a large negative number, $|x| = -x$.

Sample Problem

Find $\lim_{x\rightarrow -\infty}\frac{\sqrt{9x^{2}+2}}{2x-9}$.

Solution Walkthrough:

1. We are finding a limit at infinity, so we care about the terms with the highest power of $x$ (the dominant terms).

2. In the numerator, for very large negative values of $x$, the $+2$ is insignificant. So, $\sqrt{9x^2+2}$ behaves like $\sqrt{9x^2}$.

3. Here's the key step: $\sqrt{9x^2} = \sqrt{9}\sqrt{x^2} = 3|x|$.

4. Since the limit is to $-\infty$, $x$ is a negative number. By the definition of absolute value, if $x < 0$, then $|x| = -x$. So, the numerator behaves like $3(-x) = -3x$.

5. In the denominator, the $-9$ is insignificant, so $2x-9$ behaves like $2x$.

6. Now we can evaluate the limit of the dominant terms: $\lim_{x\rightarrow -\infty}\frac{-3x}{2x} = -\frac{3}{2}$.

Answer: -3/2.

Approximating Limits from Tables

The Key Idea

To estimate a limit $\lim_{x\rightarrow c} g(x)$, you need to see what happens to $g(x)$ as $x$ gets *arbitrarily close* to $c$ from *both sides*. A good table will show x-values that "pinch" or "squeeze" in on $c$.

Sample Problem

Given the function $g$ and asked to estimate $\lim_{x\rightarrow 4} g(x)$, three students created tables. Which one is the best for approximating the limit?

Table A (Best): 3.9, 3.99, 3.999, 4.001, 4.01, 4.1

This table correctly "squeezes" the value 4 from both sides.

Table B (Incorrect): -5, -4.5, -4.1, -4.01, -4, -3.99

This table is centered around -4, not 4.

Table C (Incomplete): 3, 3.5, 3.75, 3.85, 3.999

This table only approaches 4 from the left side. It's only good for a one-sided limit.

Solution Walkthrough:

1. The goal is to find the two-sided limit as $x \rightarrow 4$. This requires us to check values just below 4 (from the left) and just above 4 (from the right).

2. Table A shows x-values like 3.9, 3.99, 3.999 (approaching from the left) and 4.001, 4.01, 4.1 (approaching from the right). This is exactly what we need to see if the function is approaching the same y-value from both directions.

3. Table B is centered around -4, so it's irrelevant. Table C only shows values approaching from the left, so it can't be used to determine the two-sided limit.

Answer: The first table (Table A in the original problem) is the best because it shows values approaching 4 from both the left and the right.
